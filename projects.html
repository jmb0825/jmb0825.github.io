Dijkstra's Shortest Path Algorithm

This project completed in my Data Structures and Algorithms course. Using C++, I implemented Dijkstra's shortest path algorithm using a weighted, directed graph. In order to ensure O((|E|+|V|)log|V|) runtime, I implemented the min-heap data structure to be used as a priority queue. The program reads in a graph from a user supplied text file, and performs actions based on user commands. For more details and testing information, view the source code on Github.

Min-heap data structure

I implemented the min-heap data structure to later be used in the Dijsktra's shortest path project. The program reads in numbers from an input file and then relies on user commands to sort the heap, add new elemtents, and delete elements. The user can print the heap content at any time to see how the elements have been sorted. View the code on Github to test the program.
